API de Dados Abertos do SergipePrevidência

O SergipePrevidência continua a inovar, tornando o que é essencial em algo acessível: agora, os dados estão disponíveis de forma ainda mais prática. Estamos facilitando o acesso de programas de computador a essas informações. O SergipePrevidência oferece os dados do Instituto e do Finanprev do Portal da Transparência por meio de uma API (Interface de Programação de Aplicativos). Acreditamos no potencial criativo da comunidade e confiamos que ela pode desenvolver suas próprias soluções utilizando esses dados da forma que melhor atender às suas necessidades. Com isso, nosso objetivo é simplificar o processo de criação de aplicativos. Estamos abertos a parcerias com desenvolvedores interessados em aproveitar os dados do SergipePrevidência e Finanprev para criar novas ferramentas. Confira abaixo como começar.
